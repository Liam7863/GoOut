# 🎭 Афіша подій (Events Hub)

Повностековий веб-додаток для пошуку подій, перегляду деталей, збереження в обране та імітації купівлі квитків. Проєкт також включає систему персональних рекомендацій.

## 🔗 Посилання на проєкт
- **Live Demo (Frontend):** [https://go-out-seven.vercel.app]
- **API Swagger Docs (Backend):** [https://goout-api-35ch.onrender.com]

## 🛠 Технологічний стек
**Frontend:**
- React (Vite)
- React Router DOM (Маршрутизація)
- Axios (HTTP запити)

**Backend:**
- Python / FastAPI
- SQLAlchemy
- PostgreSQL 
- JWT Авторизація

**Деплой та Інфраструктура:**
- Vercel (Хостинг фронтенду)
- Render (Хостинг бекенду та хмарна БД)

## 🚀 Основний функціонал
- Реєстрація та JWT-аутентифікація користувачів.
- Перегляд актуальної афіші подій (дані збираються парсером).
- Збереження подій у персональний список "Обране" (лайки).
- Купівля квитків (симуляція транзакцій).
- Динамічний радар у профілі користувача, що оновлюється в реальному часі.
